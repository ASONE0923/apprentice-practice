## 学んだこと（新しい気付き、学び）

- bundlerとは、gemをどのバージョンで使うか一元管理できるgemのこと
- NewCops: enable とはNewCops: rubocop のバージョンをアップグレードしたときに新しいCopが自動的に有効になる設定
- shiftメソッドは、配列の先頭の要素を取り出して、その要素を削除し、その要素を返すメソッドです。例えば、以下のように使用することができます。
```ruby
array = [1, 2, 3, 4, 5]
p array.shift # => 1
p array # => [2, 3, 4, 5]
```
